# OPSD Landing (GitHub Pages)

Esta carpeta `docs/` contiene la landing page estática para descargar el APK de OPSD.

Publicación en GitHub Pages:

1. Sube el repositorio a GitHub.
2. En `Settings` → `Pages`, selecciona:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (o la rama que uses)
   - **Folder**: `/docs`
3. Guarda. En unos minutos estará disponible en `https://<tu-usuario>.github.io/<tu-repo>/`.

El botón de descarga apunta a `downloads/opsd.apk`. Si cambias el nombre o la ubicación del APK, ajusta el enlace en `docs/index.html`.
